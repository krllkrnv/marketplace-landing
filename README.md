# Marketplace Landing

Магазин цифровых товаров для геймеров (ключи, пополнения, подписки, гифт-карты).

Стек: HTML / CSS / JavaScript + Swiper. Desktop, светлая тема.

## Live

- Главная: https://krllkrnv.github.io/marketplace-landing/
- Каталог: https://krllkrnv.github.io/marketplace-landing/catalog.html

## Исходники

https://github.com/krllkrnv/marketplace-landing

## Чеклист (главная)

- [x] Баннер-карусель (autoplay, стрелки, индикаторы)
- [x] Меню «Каталог» (открытие / повторный клик / клик вне)
- [x] Переключатель валют `$ / ₸ / ₽`
- [x] Hover иконок сервисов
- [x] Hover карточек товара

## Чеклист (каталог)

- [x] Блок «Официально» — горизонтальная прокрутка (стрелка / drag), счётчик статичен
- [x] Табы категорий — только active-состояние
- [x] Фильтр слева — аккордеоны и чекбоксы (два видимых состояния)
- [x] Цена — поля от/до + dual-range, двусторонняя синхронизация, 500–1 000 000
- [x] «Применить» / «Сбросить» (сброс чекбоксов и цены)
- [x] Сортировка — dropdown, outside click, смена лейбла
- [x] Hover карточки и отдельный hover «Купить»
